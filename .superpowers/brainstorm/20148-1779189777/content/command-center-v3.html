New: Tasks Panel

Every workflow command breaks into sub-tasks with real-time status — done (✓), running (⟳ with spinner), pending (○), error (✗). Shows the actual detail of what's happening: "connecting to API...", "workspace loaded", "enabled". Plus execution time per task.

New: Processes Grid

Live view of what's running on your machine. Green = healthy, Yellow = high resource, Red = crashed, Gray = idle. Shows port numbers for servers, memory for apps. Updates in real-time.

Collapsible Sections

In the real app, each section (Tasks, Terminal, Processes) is collapsible. Click the header to expand/collapse. Only the active/relevant sections auto-expand. When idle, the overlay shrinks to just the top bar + waveform. The window auto-resizes to fit content.
