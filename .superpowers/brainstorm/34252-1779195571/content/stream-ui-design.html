Activity Stream — replaces Tasks + Terminal

One unified feed showing everything Claude does. Each stream-json event becomes a clay pill with an icon badge. 🧠 Thinking = purple, 📖 Read = jade, $_ Bash = dark with inline output, ✏️ Edit = gold, 🤖 Agent = pink. Running items pulse. Bash items show the command AND output inline — no separate terminal panel needed.

How it works: stream-json parsing

claude -p --output-format stream-json emits one JSON line per event. The Python engine parses each line, identifies the type (thinking / tool_use / tool_result / text / result), and sends a structured WebSocket message to the overlay. Each tool_use becomes a stream item with the right icon. Bash tool_results include stdout/stderr inline. The response card updates live as text blocks arrive.

What replaces what

Activity Stream replaces both the old Tasks panel AND Terminal panel. One view for everything. The response card stays separate (it's the final answer). Auto-collapse: when idle, only the input shows. When Claude starts working, the Activity Stream expands. When done, it collapses and the response card appears.

Agent spawning (future)

When Claude uses the Agent tool to spawn a subagent, it appears as a pink pill: "🤖 Agent: fix auth bug — running". Subagent activity could expand into a nested stream. But for v1, just show the top-level agent pill with status.
