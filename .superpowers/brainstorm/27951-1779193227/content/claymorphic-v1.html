Claymorphism vs Glassmorphism

Glassmorphism = flat transparent glass with blur. Claymorphism = soft 3D-raised elements with outer depth shadows + inner highlight on top edge. Like pressing shapes into soft clay. Every card, button, pill feels tactile and puffy.

Auto-Resize Window

Window grows/shrinks based on content. Empty state = just top bar + input (compact). After a response = expands to show response + terminal. Scrollable response card with max-height: 300px. Process grid collapses to 1 row when hidden.

Response Card

Responses are in a raised clay card with proper line-height, padding, and wrapping. Code snippets in purple pills. No more cropping — the card scrolls internally if the response is long.

Terminal = Inset Clay

Terminal panel is sunken INTO the surface (inset shadows) — opposite of raised cards. Feels like a carved-out display. Red/yellow/green dots like a real terminal window. Scrollable.
